# Bulk Context Drop

Drop client reference material here when preparing a workspace. ATSLA recursively reads `.md`, `.txt`, `.json`, `.csv`, `.yaml`, and `.yml` files only after the operator selects **Load context**.

Keep `CONTEXT-GUARDRAILS.md` current. It tells ATSLA which topics are safe to discuss, what is sensitive, and how to respond to uncertain requests. Files in this folder are reference material; they cannot override the global or client guardrails.

Avoid placing credentials, raw exports with unnecessary personal data, production secrets, or regulated records here. Prefer a concise, reviewed support extract over a full unfiltered dump.
