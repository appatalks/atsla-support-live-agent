# Demo Client Workspace

This is a fictional client workspace you can select from the ATSLA desktop app to explore isolated client context, bulk data, and conversation guardrails.

1. In **Client workspace**, choose **Browse** and select this folder.
2. Review `context-drop/CONTEXT-GUARDRAILS.md` before loading context.
3. Select **Load context**.
4. Start a session and ask about the demo product or support policy.

All information in this folder is fictional. Keep real client workspaces outside the repository and do not commit client data.
