# Voice References

This folder contains ATSLA voice reference WAV files.

- `appatalks-voice.wav` is the default AppaTalks reference.
- `eva-voice.wav` is the bundled Eva reference.

Add authorized reference recordings here for local use. Set `VOICE_CLONE_REFERENCE` to select an AppaTalks replacement or `EVA_VOICE_REFERENCE` to select an Eva replacement. Only add, use, or distribute recordings with the speaker's permission.
